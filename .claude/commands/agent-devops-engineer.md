# Invoke DevOps Engineer Agent

## Description
Activates the DevOps Engineer agent to handle deployment and infrastructure tasks.

## Usage
```
/agent-devops-engineer [deployment-or-infrastructure-task]
```

## Parameters
- `deployment-or-infrastructure-task`: The deployment or infrastructure task to perform

## Output
- Deployment scripts and configurations
- CI/CD pipeline setup
- Infrastructure as code
- Release notes and versioning information
- Monitoring and alerting configurations

## Example
```
/agent-devops-engineer Set up CI/CD pipeline for a Node.js application with automated testing
```
