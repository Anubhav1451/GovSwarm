# Invoke QA Engineer Agent

## Description
Activates the QA Engineer agent to test software and ensure quality.

## Usage
```
/agent-qa-engineer [feature-or-code-to-test]
```

## Parameters
- `feature-or-code-to-test`: The feature or code that needs testing

## Output
- Test plan and test cases
- Test results and reports
- Identified bugs with reproduction steps
- Quality metrics and recommendations

## Example
```
/agent-qa-engineer Test the user login functionality including edge cases
```
