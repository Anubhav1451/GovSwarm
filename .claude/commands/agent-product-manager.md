# Invoke Product Manager Agent

## Description
Activates the Product Manager agent to analyze requirements and create user stories.

## Usage
```
/agent-product-manager [requirements-or-idea]
```

## Parameters
- `requirements-or-idea`: The project requirements or idea to analyze

## Output
- Refined requirements
- User stories with acceptance criteria
- Prioritized feature list
- Product brief

## Example
```
/agent-product-manager Build a mobile app for tracking personal finances with budget categorization and bill reminders
```
