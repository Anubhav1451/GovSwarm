# Invoke Engineer Agent

## Description
Activates the Engineer agent to implement features and write code.

## Usage
```
/agent-engineer [user-stories-or-tasks]
```

## Parameters
- `user-stories-or-tasks`: The user stories or technical tasks to implement

## Output
- Implemented code
- Technical documentation
- Code comments and explanations
- Implementation approach documentation

## Example
```
/agent-engineer Create a REST API endpoint for user authentication with JWT tokens
```
